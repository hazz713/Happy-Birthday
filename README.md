# One Last Constellation

A cinematic anonymous birthday gift built with React, TypeScript, TailwindCSS, Framer Motion, and Lenis.

## Run

```bash
npm install
npm run dev
```

Then open `http://localhost:5174`.

## Structure

- `src/App.tsx` - main storytelling flow
- `src/data/chapters.ts` - copy and chapter metadata
- `src/components` - visual and interaction components
- `src/hooks/useLenis.ts` - smooth scrolling setup
- `src/styles/index.css` - Tailwind entry and custom animation system

The experience keeps the sender anonymous and avoids apology, guilt, or pressure.
